WordPress.org SVN assets (upload separately from trunk)
======================================================

These files are NOT included in the plugin ZIP. After your plugin is approved,
upload them to the `assets/` directory at the root of your WordPress.org SVN
repository (sibling to `trunk/` and `tags/`).

Required for plugin listing
---------------------------
screenshot-1.png   Frontend chat widget (recommended: 1200x900 or 4:3)
screenshot-2.png   Settings page with response rules
screenshot-3.png   Omnichannel settings section
screenshot-4.png   Go PRO feature comparison page

Optional branding
-----------------
icon-128x128.png
icon-256x256.png
banner-772x250.png
banner-1544x500.png

Tips
----
- Use PNG format.
- Avoid heavy branding or misleading UI in screenshots.
- Capture from a clean WordPress install with default theme for review clarity.
